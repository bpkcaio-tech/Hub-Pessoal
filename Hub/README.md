# HUB Pessoal

Um hub pessoal completo e responsivo com design líquido de vidro, construído em HTML5, CSS3 e JavaScript puro.

## Funcionalidades

- **Navegação por Seções**: 8 seções principais (Dashboard, Ideias, Vídeos, Edição, Demandas, Cofre, Roteiros, Perfil)
- **Design Líquido de Vidro**: Efeitos visuais modernos com backdrop-filter e sombras
- **Responsivo**: Funciona perfeitamente em desktop, tablet e mobile
- **Persistência Local**: Todos os dados são salvos no localStorage do navegador
- **Modais e Notificações**: Sistema completo de modais e toasts
- **Segurança**: PIN para acessar o Cofre
- **Kanban**: Sistema de arrastar e soltar para ideias
- **Perfil Personalizável**: Gerencie seu perfil pessoal

## Como Usar

1. Abra o arquivo `hub.html` no seu navegador
2. Navegue pelas seções usando a barra lateral ou navegação inferior (mobile)
3. Adicione conteúdo em cada seção conforme necessário
4. Todos os dados são salvos automaticamente no localStorage

## Seções

- **Dashboard**: Visão geral e estatísticas
- **Ideias**: Gerencie suas ideias com kanban
- **Vídeos**: Lista de vídeos favoritos
- **Edição**: Ferramentas de edição de texto
- **Demandas**: Lista de tarefas e demandas
- **Cofre**: Área segura com PIN
- **Roteiros**: Planejamento de roteiros
- **Perfil**: Gerencie seu perfil pessoal

## Tecnologias

- HTML5
- CSS3 (com variáveis CSS personalizadas)
- JavaScript (ES6+)
- Lucide Icons
- Google Fonts (Syne, DM Sans, Space Mono)

## Compatibilidade

Funciona offline em qualquer navegador moderno que suporte:
- CSS backdrop-filter
- ES6 Modules
- localStorage

## Desenvolvimento

Este é um arquivo único (`hub.html`) contendo todo o código necessário. Não há dependências externas além das fontes e ícones carregados via CDN.
